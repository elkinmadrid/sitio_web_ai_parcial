Definicion

El algoritmo KNN es uno de los algoritmos de clasificación más simples, incluso con tal simplicidad puede dar resultados altamente competitivos. Pertenece al dominio de aprendizaje supervisado y puede ser utilizado para el reconocimiento de patrones, extracción de datos y detección de intrusos.

DataSet

Nuestro Set de datos esta basado en información de una empresa para realizar la predicion sobre si un cliente realizaría el pago de su credito. Tenemos una variable binaria que sería el pago por defecto (Sí = 1, No = 0) (Variable Objetivo). Tambien tenemos en cuenta otras variables para realizar la predición tales como: Genero, Educación, Estado civil y edad.
